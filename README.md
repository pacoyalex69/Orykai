# Orykai Software

Web corporativa animada para Orykai Software, una consultora digital enfocada en desarrollo web, aplicaciones multiplataforma, marketing, SEO, automatización e integración de IA.

## Características

- React, TypeScript y Vite
- Animaciones con GSAP y ScrollTrigger
- Scroll suave con Lenis
- Preloader con identidad Orykai
- Galería horizontal de demos conceptuales
- Contenido centralizado en `content/siteContent.ts`
- Preparada para despliegue en Vercel

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Nota de contenido

La sección de demos muestra conceptos de laboratorio. No representa clientes reales ni métricas comerciales publicadas.
